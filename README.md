# Student Result Analysis

This project is developed for the CodeAlpha Data Analytics Internship.

## Description
The Student Result Analysis project analyzes student exam results using Python. It calculates total marks, percentage, grades, and provides basic performance insights.

## Project Structure
- code/student_result_analysis.py – Python analysis script
- data/student_results.csv – Student marks dataset

## Tools & Technologies
- Python
- Pandas

## Features
- Reads student data from CSV file
- Calculates total marks and percentage
- Assigns grades based on performance
- Displays class-level statistics

## How to Run
1. Install pandas:
   pip install pandas
2. Run the script:
   python student_result_analysis.py
